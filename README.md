# Physics Engine Simulation

A custom-built 2D Physics Engine developed from scratch as part of a university project. This project recreates the core gameplay mechanics of *Angry Birds* while focusing on real-time physics simulation, collision handling, object interaction, and projectile dynamics without relying on external physics libraries.

---

## 🎮 Overview

The main goal of this project was to explore the fundamentals of game physics and engine architecture by building a fully functional physics system completely from scratch.

The project demonstrates:

- Rigid body movement
- Gravity simulation
- Collision detection and response
- Momentum and force calculations
- Projectile physics
- Real-time object interactions

To showcase the engine in action, the repository includes an *Angry Birds*-inspired demo where the player launches projectiles to destroy structures and interact with physics-based objects.
